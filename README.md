# Zebrafish Behavioral Test Batteries Review

This repository contains the data and analysis scripts used in:

Fontana et al. — *Behavioral test batteries in adult zebrafish: current practices and methodological considerations.*

The repository provides the code required to reproduce all figures generated from the survey of zebrafish laboratories and the structured literature analysis of studies employing the Novel Tank Test (NTT).

---

## Repository structure

scripts/  
R scripts used to process the datasets and generate all figures.

data/  
Raw datasets used in the analysis.

figures/  
Output directory where figures generated by the scripts are saved.

---

## Data sources

### Laboratory survey dataset
`Forms_Data.csv`

Survey responses collected from zebrafish behavioral neuroscience laboratories describing:

- behavioral assays used
- number of tests per subject
- test order strategies
- methodological considerations

### Literature dataset
`metanalise_zebrafish_screening.xlsx`

Curated dataset of zebrafish behavioral studies identified through PubMed searches using the keywords **"zebrafish" AND "novel tank test"**, restricted to adult zebrafish studies and excluding review articles.
